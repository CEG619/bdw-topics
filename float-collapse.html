Float Collapse

Chris Gallagher

A List Apart Article

This article explains that a Float element is an object that shifts to the left or right of the page, allowing content to flow along either side. Content flows down the right side of a left-floated box and vice versa. They essentially act like "text wrap" properties for the webpage.

Float collapse happens when a parent element that contains any number of floated elements doesn't expand to completely surround those elements in the way it would if they weren't floated. Resulting in content that is visually out of order and confusing.

Smashing Magazine Article

This article talked about the background and definition of a float element. When a child element is coded with the "float" property, it can cause a visual "collapse" amongst the parent element and change the visual wrapping of the content being bounded. They make sure to say that this is not a CSS bug, and is in line with CSS specifications. The article then leads into the Clearfix problem that fixes this.
