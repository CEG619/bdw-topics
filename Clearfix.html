Clearfix: Fixing a Float Collapse

Clear is a sister property to Float. It is the solution to a "collapse" caused by the CSS positioning property of Float. When you Float something, it is no longer conccected to its parent element and the parent element fails to adjust and contain the floated element within. Clearfix should be applied to the parent element with the following css: 

              
                      .clearfix:after {
                        content: "";
                        display: table;
                        clear: both;
                      }
